# react-router-demo

这是一个入门React-Router的小巧Demo。这个demo里尽量不会为你带来新的所需知识。

##安装

### 克隆
```bash
$ git clone https://github.com/yuthelloworld/react-router-demo.git <my-project-name>
$ cd <my-project-name>
```
### 安装依赖

```bash
$ npm install
```
## 运行
```bash
npm start
```
## 浏览器查看
浏览器输入 http://localhost:8080/
